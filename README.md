Draw with Twilio
================

Try it out at
[draw-with-twilio.appspot.com](https://draw-with-twilio.appspot.com).

![Screenshot](screenshot.png)

Install
-------

First, run the following:

```
npm install
npm run build
npm start
```

Then, navigate to [localhost:9000](http://localhost:9000).

Deploy
------

```
npm run deploy
```
